얼굴 인식 과제했던거
얼굴인식 원 소스 코드 주소 URL : https://github.com/EnEhJ/EnEh

```
 장단점 분석

(1). 장점
ㄱ. PyQt를 사용하여 GUI를 구현하였기 때문에, 다양한 운영체제에서 사용 가능합니다.
ㄴ. 얼굴인식을 위해 데이터를 따로 입력할 필요가 없습니다.
ㄷ. dlib과 OpenCV를 통해 빠른 속도로 얼굴을 인식합니다.

(2). 단점
ㄱ. 주변 환경, 조명, 얼굴 크기 및 방향, 낮은 이미지 해상도 등으로 인해 얼굴인식이 제대로 동작하지 않을 수 있습니다.
ㄴ. 매 프레임마다 이미지를 리사이징하고 얼굴인식을 실행하기 때문에 많은 리소스가 사용됩니다. 
인식 주기를 설정하거나 얼굴인식이 되지 않았을 경우를 처리하는 방법을 추가하여 이러한 문제를 개선할 수 있을 것입니다.
ㄷ. 예외 처리가 부족합니다.
예를 들어, 코드를 실행하기 위해선 ”shape_predictor_68_face_landmarks.dat“ 파일을 요구하는데, 파일이 없다면 에러가 떠서 프로그램을 실행할 수 없었습니다.
다른 예로, 얼굴을 인식하지 못했을 때 프린트문으로 출력할 뿐 UI에는 피드백이 없습니다. 
```
